OSXDnDToggle
============

This is a simple script that will toggle the do not disturb feature in OS X on and off.  This works with Mavericks (10.9)

Usage
-----

./DNDToggle.sh

Notes
-----

This will toggle on do not disturb for 60 minutes or, if dnd is already on, it will turn it off.  Make sure that the script is executable!
